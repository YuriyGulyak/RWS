# RWS
### FPV Racing Wing Simulator by Unity

<br/>

#### Features:
- 10x10km terrain 
- Race wing with realistic physics
- Race track with lap timer
- Singleplayer with ghost
- Multiplayer
- Online leaderboard
- Available FPV and LOS flying
- Support all controllers (HID)
